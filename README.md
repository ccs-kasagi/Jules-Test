# テトリスゲーム

これは、ウェブブラウザで動作するシンプルなテトリスゲームです。

## ゲームの始め方

1. このリポジトリのファイル（`index.html`, `style.css`, `script.js`）を同じフォルダにダウンロードします。
2. ウェブブラウザ（Google Chrome, Firefox, Safari, Microsoft Edgeなど）で `index.html` ファイルを開きます。
3. ゲームが自動的に開始されます。

## 操作方法

- **← (左矢印キー)**: テトリミノ（ブロック）を左に移動します。
- **→ (右矢印キー)**: テトリミノを右に移動します。
- **↑ (上矢印キー)**: テトリミノを時計回りに90度回転します。
- **↓ (下矢印キー)**: テトリミノを通常より早く下に落下させます（ソフトドロップ）。
- **スペースキー**: テトリミノを一気に一番下まで落下させます（ハードドロップ）。

## ルール

- テトリミノを操作して、横一列にブロックが揃うとラインが消えます。
- ラインを消すとスコアが加算されます。
- ブロックがゲームボードの一番上まで積み上がってしまうとゲームオーバーです。
- 次に出現するテトリミノは、画面右側の「次へ」のエリアに表示されます。

## 開発環境

このゲームは以下の技術を使用して作成されています。
- HTML
- CSS
- JavaScript

特別なライブラリやフレームワークは使用していません。
